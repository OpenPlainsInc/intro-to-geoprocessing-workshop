## Local Setup

```bsah
# clone the forked repository
$ git clone YOUR_FORKED_REPO_URL

# create a virtual environment
$ python3 -m venv venv
# activate the virtual environment
$ source venv/bin/activate

# install dependencies
(venv) $ pip install -r requirements.txt

```
